# JoSAA IIT Branch Finder

Parent-friendly branch discovery tool for JoSAA opening/closing ranks.

V1 uses:

- JoSAA 2025 opening/closing ranks
- IIT institute type only
- Round 1 as the default, with rounds 1-6 available
- NIRF Engineering 2025 ordering for IITs

## Generate Data

```bash
python3 scripts/fetch_josaa_2025_iit.py
```

The script writes:

- `data/raw/josaa-2025/iit-round-*.html`
- `data/processed/cutoffs-2025.json`
- `data/processed/institutes-2025.json`
- `data/processed/cutoffs-2025.js`
- `data/processed/institutes-2025.js`

## Run Locally

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Deploy To Cloudflare Pages

This repo is ready to deploy as a static Cloudflare Pages project.

### Option 1: Cloudflare Pages UI

1. Push this repo to GitHub.
2. In Cloudflare Dashboard, create a new `Pages` project from that repo.
3. Use these settings:

```text
Framework preset: None
Build command: (leave blank)
Build output directory: .
Root directory: (leave blank)
```

4. Deploy.

### Option 2: Wrangler CLI

If you want to deploy from your machine:

```bash
npx wrangler pages deploy .
```

The repo already includes a [wrangler.toml](/Users/vishalkagde/Documents/Aahana_JEE/Josaa branch predictor/wrangler.toml) with:

- project name
- compatibility date
- Pages output directory set to `.`

### Deployment Notes

- The site is fully static. Cloudflare Pages only needs the files already in this repo.
- No server-side runtime is required for normal usage.
- If you refresh JoSAA data later, rerun:

```bash
python3 scripts/fetch_josaa_2025_iit.py
```

Then redeploy the updated static files.

### Caching

The repo includes a [_headers](/Users/vishalkagde/Documents/Aahana_JEE/Josaa branch predictor/_headers) file for Cloudflare Pages:

- `index.html` is set to revalidate on each request
- versioned CSS/JS/data assets are cacheable for a long time

This keeps reloads fast while letting new deployments go live cleanly.

## Notes

The website is static. It does not call JoSAA while parents are using the tool; it reads the normalized JSON generated by the ingestion script.

Results are historical guidance only. JoSAA closing ranks change every year and should be verified against official sources before final counselling decisions.
